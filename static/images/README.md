# Images Directory

This directory contains images used in the user-facing pages.

## Required Images

### 1. `logo.png`
- **Location**: Top of the landing page hero section
- **Recommended size**: 200x60px (or similar aspect ratio)
- **Format**: PNG with transparency preferred
- **Usage**: Company/Service logo

### 2. `promotional.jpg`
- **Location**: Right side of the landing page hero section
- **Recommended size**: 600x400px (or similar aspect ratio)
- **Format**: JPG or PNG
- **Usage**: Promotional image showcasing the service/offer

### 3. `success.png` (Optional)
- **Location**: Accepted/Confirmation page
- **Recommended size**: 80x80px
- **Format**: PNG with transparency preferred
- **Usage**: Success/confirmation icon or image

### 4. `rejected.png` (Optional)
- **Location**: Rejection page
- **Recommended size**: 80x80px
- **Format**: PNG with transparency preferred
- **Usage**: Rejection/cancellation icon or image

## Notes

- All images will gracefully hide if they don't exist (using `onerror` handler)
- Images should be optimized for web (compressed, appropriate file size)
- Consider using WebP format for better compression (update file extensions in templates if using WebP)
